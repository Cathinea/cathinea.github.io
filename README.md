## Visiting the Website

To Visit the Website, Simply [Click Here](https://cathinea.f5.si), or you can use:
- Ubuntu Terminal
```bash
  xdg-open https://cathinea.f5.si
```
- Windows Command Prompt
```bash
  explorer "https://cathinea.f5.si"
```
- MacOS Terminal
```bash
  open "https://cathinea.f5.si"
```
